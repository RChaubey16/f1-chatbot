B — Chat + Slide-out Drawer

Chat is full-width by default. A toggle button opens a standings drawer from the right.

C — Tabbed Single Page

One page with tabs: Chat and Standings. Switch between full-screen views.

Pros

  • Both views get full width
  • Simple, mobile-first

Cons

  • Can't see both at once
  • Less dynamic feel
